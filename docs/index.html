Installation

  1. Clone the repository: git clone https://github.com/M0NDsuChTiG/Void-Auditor.git
  2. Open in Android Studio and sync Gradle
  3. Connect your device via USB (ADB debugging enabled)
  4. Grant Shizuku permission (requires Shizuku app v13+)
  5. Run ./gradlew installDebug or build APK from Android Studio
Requirements: Android 7.0+ (API 24), Shizuku v13+, ADB debugging enabled, Internet for AI features (Gemini API key required in Settings).
