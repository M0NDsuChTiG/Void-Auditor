ADB STUDIO
v1.0 · KOTLIN · JETPACK COMPOSE · SHIZUKU

Mobile Security Audit & Device Management Tool

Профессиональный инструмент для аудита безопасности Android-устройств непосредственно с телефона. Работает через Shizuku API, не требует root-прав или подключения к ПК.

// FEATURES

◆ Device Security Audit — 11 diagnostic checks AUDIT
◆ AI Assistant (Gemini) — automated log analysis & IOC search AI
◆ Risk Aggregator — overall security score with fix instructions RISK
◆ App Manager — batch freeze, force-stop, clear data APPS
◆ Banking App Deep Scan — detect dangerous permissions BANK
◆ Activity Launcher — browse & launch any package activity ACTIVITY
◆ APK Backup & Restore — save and reinstall apps BACKUP
◆ Script Executor — 8 built-in audit/cleanup/lockdown presets SCRIPTS
◆ Shell Terminal — Shizuku-powered command execution SHELL
