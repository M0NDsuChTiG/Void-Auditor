Возможности

Security Dashboard

Risk Aggregator с визуальной шкалой 0-100. Сканирует ADB WiFi, Accessibility Services, SELinux, Device Admin, Overlay, banking apps. С автогенерацией fix-команд.

AI Forensics Assistant

Gemini 2.0/1.5 с контекстом устройства. Полный аудит через 11 Shizuku-команд, анализ логов на IOC, экспорт отчётов на /sdcard/.

Banking Deep Scan

Поиск финансовых приложений (Unibank, Sima, LeoPay), проверка опасных permission'ов, выявление over-privileged пакетов.

Shizuku Executor

High-level API: batch-команды, таймауты, CommandResult, полное логирование. Все screen'ы работают через единый адаптер.

APK Manager

Batch uninstall/disable/clear, backup APK, Activity Launcher — полный контроль над пакетами без root.

Terminal & Scripts

Встроенный терминал + 8 пресетов (AUDIT, LOCKDOWN, CONTROL, CLEANUP, NETWORK) с цветной категоризацией.

File Manager

Полный доступ к файловой системе через Shizuku: просмотр, копирование, удаление, переименование.

Connection Manager

WiFi ADB pairing, device info, диагностика подключения — всё в одном экране.
